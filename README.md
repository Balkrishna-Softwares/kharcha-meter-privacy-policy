# Kharcha Meter - App Website & Legal Pages

A modern, responsive, and privacy-focused landing page and legal documentation site for the **Kharcha Meter** Android Application (Daily Expense Tracker & Money Manager App).

## 🌟 Overview

This repository contains the static HTML and CSS files used to host the official product website for Kharcha Meter. It is designed with a premium, mobile-first, dark-mode SaaS aesthetic to inspire trust and drive app downloads. 

The site includes the following fully responsive pages:
* **`index.html`** - The main landing page detailing app features and the direct Play Store download link.
* **`privacy-policy.html`** - The official Data Privacy Policy.
* **`terms-and-conditions.html`** - The official Terms & Conditions.

## 🎨 Design Features

* **Premium Dark Theme:** Utilizes a deep `#0c0a09` charcoal background with subtle ambient glows (blue & peach) to create depth.
* **Glassmorphism Header:** A sticky, blurred navigation bar that includes a mobile-friendly hamburger menu overlay.
* **High Readability:** Large, punchy headings with ample line-height (`1.75`) and a soft `#a8a29e` text color, making dense legal text effortless to read.
* **Responsive Layout:** A fluid `1200px` max-width container that perfectly hugs mobile device edges without horizontal scrolling.
* **Zero Dependencies:** Built entirely with plain **HTML5** and **Vanilla CSS3**. No heavy frameworks (React, Vue, Tailwind, Bootstrap) are required, resulting in instant load times.

## 🚀 Deployment

Since the site is completely static, it can be hosted for free on any static hosting provider. It is currently perfectly optimized for **GitHub Pages**.

### Steps to Deploy via GitHub Pages:
1. Push these files to your GitHub repository (e.g., `kharcha-meter-privacy-policy`).
2. Navigate to your repository **Settings** -> **Pages**.
3. Under **Build and deployment**, select **Deploy from a branch**.
4. Choose the `main` or `master` branch and click **Save**.
5. Your modern site will be live within a few minutes!
